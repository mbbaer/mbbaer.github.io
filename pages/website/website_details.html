Web Development: Portfolio Website

This sleek website serves as a professional portfolio and web development playground. The website is fully responsive to accommodate mobile devices and widescreen monitors.

Background

These days, everyone needs a strong online presence. That's why I decided to create this website. My goal was to create a platform to showcase my projects, demonstrate my ability, and list my experience in a way that would distinguish me from others. This website is the culmination of many hours researching and implementing CSS, HTML, and Javascript in new and creative manners.

Implementation

This project traversed across every stage of the software development lifecycle (SDLC), from planning to deployment.

Role

  • Web Design: Responsible for design vision, color theme, and sourcing assets.
  • Web Developer: Responsible for implementing frontend/backend CSS/HTML/Javascript functionality.

Key Features

  • Fully Responsive: Leveraged Android SQLite to store jobs and comparison settings on mobile device. The application is tested in a clean-checkout state and with a new or cleared AVD to ensure a special environment is not needed to run the app.
  • Optimized for Performance: Wrote and executed repeatable automation tests to validate input fields, calculations, and other requirements.
  • Sleek Theme: Utilized EditText feature to enable user to enter details of the job information.
  • ScrollView: Utilized ScrollView feature to enable user seamless navigation on variety of devices.
  • Customized Comparison: Customized comparison class to enable comparing job offers based on input weights and offer details.
