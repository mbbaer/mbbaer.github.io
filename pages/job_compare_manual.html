Job Compare: User Manual

This simple application allows a user to compare job offers using information such as salary, cost of living, bonus potential, etc.

1.0 Configuration

1.1 Dependencies

  • The minimum SDK for the app is API 29: Android 10.0 (Q).
  • The app was tested on Pixel 2 API 29 and Pixel 4 API 29.

2.0 Main Features

The app has four main features the user can use to create a job comparison. These features are accessible through the Main Menu.

  • Enter or edit current job details - add or edit details of the current job
  • Enter job offers - add details of job offers to be used in a comparison
  • Adjust comparison settings - adjust the weights assigned to various comparison factors
  • Compare job offers - compares two chosen jobs

2.1 Enter / Edit a Current Job

Add or edit details of the current job.

The user must enter the following information pertaining to the current job:

  • Title - title of the job - [String]
  • Company - company of the job - [String]
  • Location - City - city where the current job is located - [String]
  • Location - State - state where the current job is located - [String]
  • Cost of living in the location - a number value representing the cost of living in the particular location - [INT] 1
  • Yearly salary - annual salary - [INT]
  • Yearly bonus - annual bonus - [INT]
  • Retirement benefits - annual retirement benefits as a percentage matched [INT]
  • Relocation stipend - dollar amount provided for covering expenses related to relocating to the job location - [INT]
  • Restricted stock award unit - dollar amount representing RSU value vested over 4 years - [INT]

2.1.1 Instructions

  1. Enter values for all fields according to input type listed above. Zero '0' can be entered for factors not provided by the current job or job offer, e.g. if the current job/ job offer does not provide any retirement benefits, the user should enter '0' for retirement benefits.
  2. Click SAVE to save information to database, or CANCEL to exit without saving.
  3. Both SAVE and CANCEL buttons return the user to the Main Menu.

Note: The user may only have one current job in the app. Once saved, the user can edit the details of the current job, but may not add another.

2.2 Enter Job Offer

Add details of job offers to be used in a comparison.

The user must enter the following information pertaining to the job offer:

  • Title - title of the job - [String]
  • Company - company of the job - [String]
  • Location - City - city where the current job is located - [String]
  • Location - State - state where the current job is located - [String]
  • Cost of living in the location - a number value representing the cost of living in the particular location - [INT]
  • Yearly salary - annual salary - [INT]
  • Yearly bonus - annual bonus - [INT]
  • Retirement benefits - annual retirement benefits as a percentage matched [INT]
  • Relocation stipend - dollar amount provided for covering expenses related to relocating to the job location - [INT]
  • Restricted stock award unit - dollar amount representing RSU value vested over 4 years - [INT]

2.2.1 Instructions

  1. Enter values for all fields according to input type listed above. Zero '0' can be entered for factors not provided by the current job or job offer, e.g. if the current job/ job offer does not provide any retirement benefits, the user should enter '0'.
  2. Click SAVE to save information to database.
  3. Click CANCEL to exit without saving, returning to the Main Menu.
  4. Click ADD NEW to add another job offer.
  5. Click COMPARE OFFER to navigate to the job comparison screen.

Note: Once the user has saved a job offer, it cannot be edited.

2.3 Adjust Comparison Settings

Adjust the weights assigned to various comparison factors.

These weights are used in the Job Score calculation to take into account the varying degree of importance a user might assign to each factor. A factor with a higher weight value will place more importance on that factor compared to a factor with a lower weight value.

  • Salary Weight - adjust the importance of the Salary in the job score calculation - [INT]
  • Bonus Weight - adjust the importance of the Bonus in the job score calculation - [INT]
  • Retirement Benefits Weight - adjust the importance of the Retirement Benefits in the job score calculation - [INT]
  • Relocation Weight - adjust the importance of the Relocation Stipend in the job score calculation - [INT]
  • RSUA Weight - adjust the importance of the RSUA in the job score calculation - [INT]

2.3.1 Instructions

  1. Enter values for all fields according to input type listed above. Zero '0' can be entered if the user does not want a particular factor to be considered in the job score calculation, e.g. if the user places no importance on the RSUA and does not want that factor considered when ranking jobs, the user should enter '0' for the RSUA Weight.
  2. Click SAVE to save weights into the database.

Note: If no weights are assigned, all factors are considered equal, and a default weight of '1' is assigned to each factor. The user cannot assign a negative value to a factor. The user can assign '0' to individual factors, but cannot assign '0' to all factors.

2.4 Compare Jobs

Compares two chosen jobs.

2.4.1 Instructions

  1. Review job details list for accuracy. The list displays jobs sorted by job score descending.
  2. Select two jobs to compare.
  3. Click COMPARE SELECTED JOBS to see job details for the selected jobs.
  4. Conduct another comparison, if necessary, by clicking RETURN.

Note: A current job and at least one job offer must be saved to database in order to perform the job comparison.

Appendix

  • The calculation for Adjusted Salary for Cost of Living that is used in the Compare Job Offers details is as follows: aSal = Salary / (COL/100)
  • The calculation for Adjusted Bonus for Cost of Living that is used in the Compare Job Offers details is as follows: aBonus = Bonus / (COL/100)
  • The calculation for job score is as follows: Job Score = AYS + AYB + RS + (RPB * AYS / 100) + (RSUA / 4)
    • AYS = Adjusted Yearly Salary
    • AYB = Adjusted Yearly Bonus
    • RBP = Retirement Benefit Percentage
    • RS = Relocation Stipend
    • RSUA = Restricted Stock Unit Award

For example, if the weights are 2 for the yearly salary, 2 for relocation stipend, and 1 for all other factors, the score would be computed as:
Job Score = (2/7 AYS) + (1/7 AYB) + (2/7 RS) + (1/7 (RPB AYS / 100)) + (1/7 (RSUA / 4))
